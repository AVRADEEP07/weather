# Weather App 🌤️

A weather app built with HTML, CSS and JavaScript using OpenWeatherMap API.

## Features
- Real-time weather data
- Forecast information
- Weather for major cities worldwide

## Setup
1. Clone this repo
2. Copy `config.example.js` to `config.js`
3. Add your OpenWeatherMap API key in `config.js`
4. Open `index.html` in browser

## Technologies
- HTML, CSS, JavaScript
- OpenWeatherMap API
- Bootstrap
